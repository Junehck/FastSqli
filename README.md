# FastSqli

单页mysql注入环境自带数据仅需填账号密码无需其他复杂配置开箱即用。
自定义过滤/拦截规则进行bypass等测试，支持报错/联合/延时/布尔/时间/堆叠 的数值或字符注入

# 使用演示

* 报错
* 布尔
* 时间
* 联合
* 堆叠
